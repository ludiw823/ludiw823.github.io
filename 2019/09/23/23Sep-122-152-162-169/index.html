162 Find Peak Element

基本思想是用二分法查找，如果是升序列，就往右边找，如果是降序列，就往左边找。
mid = lo + (hi - lo)/2 根据整除的性质 nums[mid+1]肯定是存在的，所以二分法查找的时候只比较nums[mid]和nums[mid+1]。

169 Majority Element
